**Блок практических работ по предемету Data science**

### Блок «Методы машинного обучения» 
1.  **phds_hw1_bio(1).ipynb (ДЗ №1)**
Практическая работа по теории вероятностей. В нем основное внимание уделяется расчету математического ожидания и дисперсии для случайных величин, возникающих в биологических экспериментах, а также работе с экспоненциальным распределением.

2.  **bm_linreg_hw(1).ipynb (ДЗ №2)**
    Практическая работа по линейной регрессии. Основной упор сделан на понимание процесса оптимизации: как изменение шага обучения (learning rate) влияет на то, сможет ли модель правильно подобрать коэффициенты для предсказания признаков.

3.  **bm_logreg_hw(1)(1).ipynb (ДЗ №3)**
    Практическая работа по логистической регрессии (классификации). Здесь сравниваются обычный градиентный спуск и его стохастический вариант (SGD), а также анализируется поведение функции потерь (loss) на тренировочных и тестовых данных.

4.  **bm_reg(1).ipynb (ДЗ №4)**
    Практическая работа: методы регуляризации регрессионных моделей. Он содержит задачи на предотвращение переобучения моделей.
    
### Блок «Математическая статистика» 

1.  **HW1_stats_2026_bio(2).ipynb (ДЗ №5)**
    Практическая работа по методам точечного оценивания параметров. Здесь разбираются свойства оценок (несмещенность, состоятельность) и их применение к распределениям, которые часто моделируют биологические процессы (например, распределение Пуассона).

2.  **HW2_stats_2026_bio(1).ipynb (ДЗ №6)**
    Практическая работа по асимптотической теории. Сравнивается эффективность различных статистик на больших выборках и используют усеченное среднее (trim_mean) для повышения устойчивости оценок к выбросам в данных.

3.  **HW3_stats_2026_bio(1).ipynb (ДЗ №7)**
    Практическая работа по проверке статистических гипотез. Рассматривается классический t-критерий и статистика Вальда, а также анализируется, насколько сильно различаются результаты этих методов на малых и больших выборках.

4.  **HW4_stats_bio_2026(1).ipynb (ДЗ №8)**
    Практическая работа: мощность статистического теста (способность найти различия, если они есть). С помощью симуляций Монте-Карло проверяется, насколько теоретические формулы соответствуют реальности при планировании биологического эксперимента.

5.  **hw5_stats_biology_2026(1).ipynb (ДЗ №9)**
    Практическая работа по построению доверительных интервалов. В нем исследуется реальный уровень доверия — то есть то, как часто интервал на самом деле накрывает истинное значение параметра при разных размерах выборки.
